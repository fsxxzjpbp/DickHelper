# 旧版数据迁移指南

> 新版 DickHelper 不再基于浏览器运行，而是独立的桌面应用 (Electron)。因此旧版存储在浏览器中的数据无法自动读取。**你需要先从旧版手动导出数据**，再到新版中导入。

---

## 操作步骤

### 1. 打开旧版 DickHelper

访问旧版 DickHelper 网站。如果你用的是本地部署版本，启动本地服务；如果用的是在线 Demo，打开 https://dick.juwo.my 。

### 2. 导出数据

1. 在页面「主页面」标签下，找到 **记录新的手艺活** 卡片。
2. 滚动到卡片底部，点击 **「导出数据」**  按钮。
3. 浏览器会下载一个名为 `masturbation_records.json` 的文件。
4. **保存好这个文件**，后续新版将通过它导入数据。

> 如果看不到「导出数据」按钮，请确认你使用的是 v0.1.0 及以上版本。更早的版本不支持导出功能。

### 3. 备份建议

- 把 `masturbation_records.json` 复制到云盘或 U 盘，以免误删。
- 该文件是明文 JSON，可以用记事本打开查看内容，但**不要修改**，否则可能导致导入失败。

### 4. 导入到新版

新版发布后，在应用内找到「导入数据」功能，选择你导出的 `masturbation_records.json` 文件即可完成迁移。

## 常见问题

**Q: 我没法访问旧版网站了怎么办？**

如果旧版网页已经无法打开，但你之前导出过 JSON 文件，直接用文件导入即可。如果既没有 JSON 文件也无法打开旧版，数据无法恢复。

**Q: 旧版数据还在浏览器里，但网站打不开了？**

旧版数据存储在浏览器的 localStorage 中，存储在哪个**域名**下就在哪个域名下。如果你用的是在线 Demo (dick.juwo.my)，数据只能通过该域名访问。如果你用的是本地部署 (localhost)，数据在本地浏览器中。你可以尝试：

1. 重新部署一份旧版到同样的域名
2. 打开网站，按上面的步骤导出数据

**Q: 导出文件里有什么敏感信息？**

文件中包含：记录时间、持续时长、备注（如有）。这是你自己的数据，请妥善保管，**不要发给别人**。

**Q: 新版什么时候发布？**

请关注 GitHub Release 页面。
